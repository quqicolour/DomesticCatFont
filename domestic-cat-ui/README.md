# DomesticCatFont — NFT Frontend

Web3 dApp frontend for minting and managing DomesticCat NFTs on ARC Testnet.

Built with: **React 18** · **Vite 5** · **TypeScript** · **wagmi v2** · **RainbowKit v2** · **viem v2**

## Tech Stack

| Layer | Choice | Why |
|-------|--------|-----|
| Framework | React 18 + Vite | Fast HMR, TypeScript first |
| Web3 | wagmi v2 + viem v2 | Modular, tree-shakeable Ethereum lib |
| Wallet | RainbowKit v2 | Best-in-class wallet connection UX |
| Styling | Plain CSS + CSS Variables | Zero-runtime, full control |
| Chain | ARC Testnet (5042002) | Custom EVM L2 |

## Architecture

### State & Data Flow

```
useAccount()              → wallet address
  ↓
balanceOf (wagmi multicall3)
  ↓
ownerOf scan [0..99]  ── wagmi multicall3 ──→ 100 results (1 RPC)
  ↓
filter by address
  ↓
[tokenURI, getNFTPowerLevel, tokenURI, getNFTPowerLevel, ...]  ── wagmi multicall3 ──→ 2N results (1 RPC)
  ↓
decode base64 SVG → renderCatImage() → dangerouslySetInnerHTML
```

Total RPC calls for My Cats with N cats: **2** (both multicall3 batches).

### Why `ownerOf` Scan Instead of `tokenOfOwnerByIndex`?

`DomesticCatNFT` does **not** implement ERC721Enumerable (`tokenOfOwnerByIndex` is absent from the ABI). NFT enumeration uses `ownerOf(i)` for token indices 0–99 — a reasonable scan range for a 10,000-cap collection.

### SVG Rendering

Token metadata is stored entirely on-chain as base64-encoded JSON. The SVG is generated by `CatSVGRegistry.generateSVG()`, encoded as `data:image/svg+xml;base64,...`, and stored as the `image` field. The frontend decodes it client-side with `atob()` and renders via `dangerouslySetInnerHTML`.

> ⚠️ ABI-encoded `string` returns from Solidity may contain padding bytes (`\x00`, `\x16`, etc.) before the actual content. The `decodeTokenUri()` utility strips these before parsing.

## Troubleshooting

### "Connect Wallet" button not showing chains
Make sure `wagmi.ts` chain config matches the active network in MetaMask. The app auto-switches to ARC Testnet (chainId `5042002`) on load.

### SVG images not loading on My Cats page
The contract's `tokenURI` returns ABI-encoded strings with control-character padding. Ensure `decodeTokenUri()` in `MyCatsPage.tsx` is stripping `\x00`–`\x1f` bytes before locating the `data:` prefix.

### Transaction reverted when minting
- Verify MetaMask is on **ARC Testnet** (chainId `5042002`).
- Confirm the wallet has sufficient native token balance to cover `mintFee × quantity`.
- Check that the mint quantity does not exceed your remaining quota (max **5 per address**).

### RPC rate limits
If you encounter rate-limiting on the public RPC, set a private RPC URL in `src/config/wagmi.ts`:
```ts
export const publicClient = createPublicClient({
  chain: ARC_CHAIN,
  transport: http('https://your-private-rpc.io'),
})
```
